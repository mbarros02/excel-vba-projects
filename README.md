# Projetos com VBA

Este repositório contém uma coleção de ferramentas e automações desenvolvidas em VBA (Excel, Word e integrações) para otimizar processos administrativos, financeiros e de BI.

## 🛠️ Tecnologias e Habilidades
- **Linguagens:** VBA (Excel, Word, Outlook), SQL.
- **Integrações:** Automação de Office (Interop), Conexão com Bancos de Dados (ADO/DAO).
- **Processos:** Automação de workflow, Extração de dados (ETL), Geração de Dashboards.

## 🚀 Como utilizar
Como os códigos VBA estão em arquivos binários (.xlsm), recomendo olhar a pasta `/src` de cada projeto para visualizar a lógica nos arquivos `.bas`.
